## 服务模块
- mall-dependency：项目版本统一管理模块
- mall-admin：商城管理台服务
- mall-gateway：商城网关模块
- mall-common：公用模块
 - mybatis-plus-generator-ui：mybaits-plus代码生成器
- mall-auth: 商城认证模块
- mall-Monitor：商城监视器模块
